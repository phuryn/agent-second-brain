# [PLATFORM] False Beliefs

This file is an evidence log of things that seem like they should hurt performance — but don't. Or things you were told were rules — but aren't.

This is not a list of "myths." It's a personal evidence log. Every entry should have a specific data point behind it.

---

## How to Use This File

Load this when:
- A result surprises you
- You're about to enforce a "rule" and want to check if it's actually true
- A piece performs differently than expected

Add to this file when:
- A hypothesis is rejected (it turned out not to matter)
- You see data that contradicts something you believed
- A piece performs well despite "breaking a rule"

## Notes

- "No data" is not the same as "false." Only add entries here when you have actual evidence against the belief.
- A false belief that's true in some contexts can coexist with a rule that's true in most contexts. Note the conditions.
- When a false belief here contradicts an active rule in `rules.md`, that's a signal the rule needs review.
