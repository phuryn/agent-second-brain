# [PLATFORM] Templates

Templates are starting structures, not rigid scripts. Use them to eliminate blank-page friction and front-load structural decisions. Deviate when the content demands it — but know which template you started from.

---

## How to Use These Templates

1. Identify the content type (what is this piece trying to do?)
2. Select the template that matches
3. Fill in the structure
4. Run `craft/draft-gate.md` before publishing

## Adding New Templates

Before adding a new template:
- Has this structure produced good results at least twice?
- Is it meaningfully different from existing templates?
- Can you name exactly when to use it (and when not to)?

If yes to all three, add it here with a new ID and update the index.
