# Rejected Hypotheses — Immune Memory

This file stores beliefs that were tested and proven wrong. Without this, the same false belief gets rediscovered and re-tested every few months.

**When to add an entry:** When a hypothesis is rejected (see `index.md` rejection criteria), move it here with the evidence that killed it. Never delete — archive is cheap, re-testing is expensive.

**When to read this file:** At the start of any task where a known false belief might resurface. The agent should check this file proactively.

---

## Entry Format

```
### [HYP-ID] — [Short description of the belief]
**Was:** [The original hypothesis statement]
**Evidence against:** [What disproved it]
**Key learning:** [What the failure taught — often more valuable than the hypothesis itself]
**Replaced by:** [If a better rule emerged, link it here]
**Rejected:** [DATE]
```
